
This directory contains a subset of cygwin commands.

Cygwin command can run out of a cygwin installation.

Attention:
To avoid conflicts, all commands MUST be from identical snapshot.

To get the date of the current snapshot, please use:

> uname -a
> CYGWIN_NT-6.1 VIRCVO11606NBH 2.10.0(0.325/5/3) 2018-02-02 15:16 x86_64 Cygwin

This means snaphot is from February 2018.
